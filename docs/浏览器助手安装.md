# 号池监控浏览器助手安装

浏览器助手用于从 Chrome 或 Edge 已经登录的 New API 站点读取会话，随后自动填入号池监控。每次只读取渠道向导第一步填写的那个地址，不扫描其他站点。安装一次后，新增或更新渠道时可以直接点击“一键读取当前地址”。

## 安装

1. 在 New API 渠道的“网页登录”区域点击“下载浏览器助手”。
2. 解压下载的压缩包，保留解压后的整个文件夹。
3. Chrome 打开 `chrome://extensions`，Edge 打开 `edge://extensions`。
4. 开启页面中的“开发者模式”。
5. 点击“加载已解压的扩展程序”，选择刚才解压出的文件夹。
6. 刷新号池监控页面，页面显示“浏览器助手已连接”即表示安装完成。

## 一键读取

1. 先在浏览器中登录需要监控的 New API 站点。
2. 回到号池监控的渠道向导，选择“网页授权”。
3. 点击“一键读取当前地址”。
4. 页面显示读取成功后继续设置阈值并保存渠道。

如果当前填写地址尚未登录，浏览器助手会打开该地址。完成登录后回到号池监控，再点击一次“一键读取当前地址”。

## 自定义号池监控地址

浏览器助手默认信任 `https://jiance.zzlye.xyz`、本机默认地址以及本机 `8080` 端口。使用其他域名或端口时，在扩展程序详情中打开“扩展程序选项”，每行填写一个号池监控地址，然后刷新号池监控页面。公网号池监控地址必须使用 HTTPS，第一行地址会作为点击助手图标时默认打开的页面。
