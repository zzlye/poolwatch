# 号池监控

## 简介

号池监控用于集中查看不同渠道的余额、额度和账号状态。余额不足、账号异常或渠道无法访问时，可及时向已启用通知的电脑和手机发送提醒。

## 如何使用

1. 首次打开后，使用初始化口令创建管理员账号。
2. 登录后进入“渠道”，按向导填写渠道地址、登录信息、关注额度和提醒阈值。CLIProxyAPI 渠道填写管理密钥后即可读取账号状态。
   如果渠道只支持 GitHub、Linux.do 等网页登录，可选择“网页授权”；安卓应用会打开独立登录窗口，电脑端也可手工导入登录结果。
3. 测试连接成功后保存渠道，随后可在“总览”查看最新状态。
4. 在“推送设备”中允许通知，并发送测试通知确认电脑或手机能够收到提醒。
5. 需要补充余额时，点击渠道上的“充值”前往该渠道的充值页面。
6. 安卓手机可从项目发布页面下载最新安装包，安装后登录并开启“持续监控”。

## 功能介绍

- 集中查看 New API、Sub2API、chatgpt2api、CLIProxyAPI 和自定义渠道。
- 支持 New API 与 Sub2API 的 GitHub、Linux.do 等网页登录方式。
- 分别设置钱包余额、订阅额度和图片生成次数的提醒阈值。
- 查看渠道当前状态、历史变化、异常记录和恢复情况。
- 查看 chatgpt2api 中经过隐藏处理的账号状态和剩余额度。
- 查看 CLIProxyAPI 中经过隐藏处理的账号提供商、可用状态、请求统计和恢复时间，并按账号数量发送提醒。
- 在电脑与安卓手机上接收余额不足、登录失效和渠道异常通知。
- 安卓应用支持持续监控、测试通知和后台运行设置引导。
- 自定义渠道名称、检查频率、充值页面和通知设备。
